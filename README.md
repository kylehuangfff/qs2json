qs2json 复杂结构的查询字符串到JSON对象的转换
====

## 安装

```
npm install qs2json --save
```

## 示例
```
    qs2json('a=1&b=2');
    //=> {"a":"1","b":"2"}

    qs2json('a.p1=1&a.p2=2');
    //=> {"a":{"p1":"1","p2":"2"}}
    
    qs2json('a[]=1&a[]=2');
    //=> {"a":["1","2"]}
    
    qs2json('a.b.c.d.e=1');
    //=> {"a":{"b":{"c":{"d":{"e":"1"}}}}}
    
    qs2json('a.b.arr[]=1&a.b.arr[]=2');
    //=> {"a":{"b":{"arr":["1","2"]}}}
```
